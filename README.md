# j-code-ext-typescript

A JCode **language** extension for TypeScript: coding suggestions (snippets), a
basic formatter (Prettier, best-effort), and helpers. Installed through the
[JCode Marketplace](https://github.com/janrick123/j-code-marketplace).

See [`extension.yaml`](extension.yaml) for the manifest, and
[j-code-ext-csharp](https://github.com/janrick123/j-code-ext-csharp) for the
`language` manifest schema.
